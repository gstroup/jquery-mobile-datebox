Forced Popup Mode

You can force the dialog mode off with 'useDialogForceFalse' set to 'true'

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useDialogForceFalse": true}'>

Forced Dialog Mode

You can force the dialog mode on with 'useDialogForceTrue' set to 'true'

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useDialogForceTrue": true}'>

Inline Mode

You can force DateBox to display inline with 'useInline' set to 'true'.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useInline": true}'>
