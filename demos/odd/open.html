No Button, Focus Mode

You can open on focus, removing the button by using 'noButtonFocusMode' set to 'true'

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"noButtonFocusMode": true}'>

Open on Link

You can force open the dialog by clicking here. Also shown without a button (noButton : true)

HTML 
<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"noButton": true}'>
   
<a href="#" id="linkmodelink">Open Link</a>
jQuery 
$('#linkmodelink').live('click', function() {
  $('#mydate').datebox('open');
});
